Quiz:React Native

1. What is React?.

React Native React Js is a Javascript Library where you can develop and run faster web applications.

2. What are the advantages of React Native?

Advantages of React Native 1.Highly useful for native app developers with benefitting features. 2.Superbly cost-effective and code reuse. 3.No need for recompilation with changes and live reloading of codes. 4.Use of intelligent debugging tools for text editors.

3. What language does React Native use?

It is based on React js, which is written in JavaScript.

4. What is HOC in ReactNative

Higher-Order-Component, shortly known as HOC is an advanced React Native technique to reuse the component logic. The function obtains a component and returns a new element.

5. What is the use of arrow function in ReactNative

The use of Arrow functions in React Native is to help reduce your application's memory consumption. It is done by reducing the CPU time required to iterate the over loops to generate the components necessary for your lists.
